Exquisite Corpse

Description

A collaborative p5.js project where each teammate designed a full character and then swapped modular body parts to create hybrid figures. I designed Amani, a character built from simple geometric primitives inspired by Malika Favre's bold, high-contrast style.

Each body part was created as a reusable, parameterized function so it could stand alone and mix cleanly with others. The final result is a playful human–alien–robot hybrid that highlights modular design, creative coding as a system, and the visual surprises that emerge from collaboration.
