Face Generator

Description

An interactive generative face system built with p5.js that explores how simple geometric forms can produce expressive variation. Mouse position controls the face's size, color, petal count, and eye complexity, while a toggleable random mode generates stable, unpredictable variations on demand.

The project emphasizes playful interaction, modular design, and the creative potential of minimal shapes.
